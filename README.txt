HSN Tax Calculator
=================

A simple application to calculate GST based on HSN codes.

Installation
-----------
1. Extract all files from the ZIP archive
2. Double-click HSN_Tax_Calculator.exe to run the application

Features
--------
- HSN code selection
- Tax calculation (CGST, SGST)
- Save entries to CSV
- Microsoft OneDrive integration

Troubleshooting
--------------
- If you see "Windows protected your PC" message, click "More info" then "Run anyway"
- If the application doesn't start, install Visual C++ Redistributable from Microsoft's website
- For any other issues, contact support
